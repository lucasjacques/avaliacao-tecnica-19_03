## About The Project

* This repository contains the implementation of a Technical Assessment for a Senior QA Tester position, covering a comprehensive set of quality assurance practices, including manual testing, test automation, API validation, and fundamental database queries.

* The objective of this project is to demonstrate a well-rounded QA skill set, combining exploratory thinking, automation capabilities, API testing, and theoretical knowledge, all supported by clear and structured documentation.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Markdown Reader (VS Code recommended)
  * I strongly recommend using VS Code with the "Preview" feature ("ctrl+shift+v" shortcut) because other applications may not load the videos properly

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lucasjacques/avaliacao-tecnica-19_03.git
   ```
2. Open it on VS Code
    * Open the project repository on VS Code and navigate to the files detailed below to evaluate each subject of the Technical Assessment (detailed in Usage topic below)
